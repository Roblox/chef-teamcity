# Description

Installs TeamCity server and agent

# Requirements

## Platform:

* mac_os_x
* ubuntu
* centos

## Cookbooks:

* ark (~> 3.0)
* homebrew (~> 4.2)
* java (~> 4.0)
* runit (~> 4.0)
* seven_zip
* systemd

# Recipes

* teamcity::default - Installs TeamCity Agent on target OS
* teamcity::server - Installs TeamCity Server on CentOS or Ubuntu
